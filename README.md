# chat_client
Chat client for office use. Created with Qt Creator, with QML and Widgets elements.

Данный клиент реализует клиентскую часть офисного чата, позволяет входить в систему, регистрировать аккаунт,
обмениваться сообщениями и файлами, а также формировать и выводить списки задач, в соответсвии с требованиями заказчика.
Данный проект был первым для меня в фреймворке QT, поэтому не отличается изящностью, однако позволил хорошо с ним познакомиться.
